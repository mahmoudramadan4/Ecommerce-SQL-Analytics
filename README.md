 E-commerce SQL Analytics Project

 Overview

This project simulates an e-commerce database and performs analytical queries using SQL.

 Database Schema

* Customers
* Products
* Orders
* Order Items
* Payments

Objectives

* Analyze customer behavior
* Calculate total revenue
* Identify top-selling products
* Segment customers based on activity
  
 Key Queries

* Top 3 customers by spending
* Orders above average value
* Monthly revenue analysis
* Customer segmentation using CASE
* Top selling products

Tools Used

* SQL Server
* T-SQL

 How to Run

1. Run `01_create_tables.sql`
2. Run `02_insert_data.sql`
3. Run `03_queries.sql`

 Sample Insights

* Identify high-value customers
* Track revenue trends
* Discover best-selling products
